Open-source project / OSS

Cherryを一緒に育てる。

Cherryはオープンソースのβ版ToDoアプリです。まだ完成品ではありません。だからこそ、バグ報告、要望、設計への意見、ドキュメント改善、コード貢献のすべてが大きな助けになります。

Cherry is an open-source beta todo app. Feedback, bug reports, feature ideas, documentation improvements, and code contributions are welcome.

Report an issue View repository

How to contribute

コードを書かなくても、貢献できます。

01

バグ報告

表示崩れ、操作できない場所、分かりにくい挙動などをIssueで教えてください。スクリーンショットがあると助かります。

02

機能要望

「こう使いたい」「ここが惜しい」という意見は、Cherryの方向性を決める材料になります。

03

コード貢献

小さな修正、UI改善、リファクタリング、ドキュメント追加など、できる範囲のPull Requestを歓迎します。

What Cherry needs

今ほしい貢献

UI / UX feedback

モバイルや横長画面での使いやすさ、見た目の分かりやすさについての意見。

Bug reports

タスク作成、線の接続、日付レーン、編集、保存まわりの不具合報告。

Documentation

README、使い方、開発環境、貢献方法の整理。

Example tasks

Cherryの紹介に使える、分かりやすいサンプルタスクやデモ構成。

Beta status

Cherryはまだβ版です。

仕様やUIは今後変わる可能性があります。正式版に向けて、まずは「使っていて迷うところ」「良いと思ったところ」「壊れているところ」を集めたいです。

Cherry is still in beta. The UI and behavior may change. The most valuable contribution right now is clear feedback from real usage.
