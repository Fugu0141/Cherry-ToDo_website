Open-source community

Cherryを一緒に育てる。

Cherryは、タスクを流れで整理するオープンソースのβ版ToDoアプリです。まだ完成品ではないからこそ、使ってくれる人のフィードバック、Issue、Pull Requestがそのままプロダクトの力になります。

Cherry is an open-source beta todo app. Feedback, bug reports, feature ideas, documentation improvements, and code contributions are welcome.

What helps Cherry?
  • バグや違和感の報告
  • 欲しい機能・改善案
  • READMEや使い方の改善
  • 小さなコード修正やPull Request

Start here

コードを書かなくても、貢献できます。

01

Issueを書く

操作中に困ったところ、表示崩れ、分かりにくい挙動を教えてください。短い報告でも助かります。

02

アイデアを出す

「こう使いたい」「この導線が欲しい」など、実際の使い方から出た要望を歓迎します。

03

Pull Requestする

小さな修正、UI改善、ドキュメント追加、リファクタリングなど、できる範囲で参加できます。

Beta status
